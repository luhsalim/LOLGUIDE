O que é o LOLGUIDE?

LOLGUIDE é um projeto que nasceu durante a Imersão Dev da Alura, com o objetivo de ser um guia completo para os jogadores de League of Legends. Aqui, você pode pesquisar por campeões, descobrir informações detalhadas sobre suas habilidades, rotas preferenciais e builds otimizadas para diferentes momentos do jogo. Além disso, a página destaca campeões populares e os mais fortes do meta atual, permitindo uma navegação fácil e informativa.

Por que criei este projeto?

Este projeto foi desenvolvido como parte do desafio proposto pela Imersão Dev da Alura. A Imersão é um evento educacional que permite aos participantes mergulharem em desenvolvimento web, aplicando conhecimentos de HTML, CSS e JavaScript na construção de projetos práticos. Como sou um grande fã de League of Legends, decidi unir minha paixão pelo jogo com o aprendizado de desenvolvimento web, criando o LOLGUIDE.

Minha experiência na Imersão Dev

Participar da Imersão Dev foi uma experiência enriquecedora. Aprendi a transformar uma ideia em um projeto concreto, desenvolvendo habilidades essenciais para a carreira de desenvolvedor front-end. Utilizei tecnologias como HTML5, CSS3 e JavaScript para construir um site responsivo e interativo, e o evento da Alura foi fundamental para meu crescimento nessa jornada.
